2. the 525th byte (start counting bytes at 1) in the 1st file in the root directory
logical cluster (little endian) tells you where the file is located. if you check the FAT it tells you where the next logical cluster is 
add 31 to map from the cluster number to sector number 
the file 14.txt first logical cluster is 05 00 (little endian) -> 0005
each cluster is 512, so we want 525-512 = 13th in the next logical cluster
FAT tells us FF0 FFF FFF 004 FFF 006 for the first 6, 006 is the next 
(31+6)*512 is the start and then count up 12 bytes (because we start at 1 and 4A00 is 1)
4A0C = 38
the answer is 38

go to (0x2F+0d31)*0d512 = 9c00

4. the first logical cluster of a deleted file (any file)
