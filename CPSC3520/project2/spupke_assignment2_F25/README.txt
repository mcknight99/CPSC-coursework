assignment2.ml

   contains - helper function to check if a list contains an item
   first_duplicate - Find and return the first duplicate in an integer list
   sumcheck - helper function to check if any elements in list b sum with anum to v
   sumOfTwo - Look for a pair of elements from two integer lists that sum to a given value.
                Two arrays contain numbers able to produce a given sum.
   track - helper function that tracks unique values between lists in order of the first list
   list_intersection - Computes the intersection of two lists as a set (no duplicates), preserving the
                order in which elements first appear in the first list.
   unique - Remove duplicates from a list while preserving the order of first appearances.
   add_head - add item to the start of each list in lists
   join_lists - "concatenates" two lists into one list, preserving order
   powerset - Takes a set S represented as a list and returns the set of all subsets of S

    The file uses only pure functional programming, no imperative features.
    There are no loops, local or global variables, and no array module or sequential logic.
